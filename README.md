
# Jonas Makeke - Portfolio Professionnel

Portfolio moderne et responsive conçu avec une approche orientée résolution de problèmes, mettant en valeur mes compétences techniques, mes projets et mon expérience en développement web full-stack.

## 🎯 À Propos

Portfolio personnel de Jonas Makeke, développeur full-stack et créateur de contenu spécialisé dans la transformation de problèmes complexes en solutions élégantes. Ce site présente une interface utilisateur optimisée pour les résolveurs de problèmes, avec des éléments visuels clairs et des informations structurées.

## ✨ Fonctionnalités

### Design & Interface
- 🎨 **Design Moderne** : Interface claire et professionnelle orientée résolution de problèmes
- 🌓 **Mode Sombre/Clair** : Basculement de thème avec sauvegarde de préférence
- 📱 **100% Responsive** : Optimisé pour tous les appareils (mobile, tablette, desktop)
- ⚡ **Performances** : Site statique léger sans dépendances de frameworks
- ✨ **Animations Fluides** : Transitions et effets visuels soignés

### Fonctionnalités Interactives
- ⌨️ **Animation de Typing** : Texte animé dans la section hero
- 🎠 **Carousel d'Images** : Galerie avec défilement automatique et navigation
- 🎯 **Scroll Animé** : Sections qui s'animent au défilement
- 🧭 **Navigation Fluide** : Navigation sticky avec indicateur actif
- 📧 **Formulaire de Contact** : Formulaire fonctionnel avec validation

### Technologies & Compétences
- 🎨 **Icônes Technologiques** : Icônes SVG avec couleurs spécifiques pour chaque technologie
- 📊 **Barres de Progression** : Visualisation des compétences avec couleurs personnalisées
- 🏷️ **Tags de Projets** : Badges colorés avec icônes pour chaque technologie utilisée

## 📋 Sections du Portfolio

1. **Hero** : Introduction avec animation de typing et image de profil
2. **About** : Présentation personnelle avec statistiques (projets, expérience, taux de succès)
3. **Methodology** : Approche de résolution de problèmes en 4 étapes (Analyze, Design, Implement, Optimize)
4. **Projects** : Présentation de projets avec tags technologiques et liens
5. **Skills** : Compétences techniques avec barres de progression colorées par technologie
6. **Gallery** : Carousel d'images avec navigation et indicateurs
7. **Contact** : Formulaire de contact avec email et téléphone
8. **Social** : Réseaux sociaux et plateformes de contenu

## 🛠️ Technologies Utilisées

### Frontend
- **HTML5** : Structure sémantique
- **CSS3** : Design moderne avec variables CSS et animations
- **JavaScript (ES6+)** : Interactivité et fonctionnalités dynamiques

### Technologies Affichées
- React, TypeScript, CSS, JavaScript, Vue.js
- Node.js, Python, Django, PostgreSQL, MongoDB
- Git, Docker, AWS, Figma, Webpack, Sass, Jest

## 🚀 Installation et Utilisation

### Prérequis
Aucun prérequis nécessaire ! C'est un site statique qui fonctionne directement dans un navigateur.

### Installation Rapide

1. Cloner ou télécharger le repository
```bash
git clone https://github.com/jonas-makeke/portfolio.git
cd portfolio
```

2. Ouvrir `index.html` dans votre navigateur web
3. C'est tout ! 🎉

### Développement Local

Pour une meilleure expérience de développement, utilisez un serveur local :

#### Avec Python
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

#### Avec Node.js
```bash
npx http-server
```

#### Avec VS Code
Installer l'extension "Live Server" et cliquer sur "Go Live"

Puis ouvrir `http://localhost:8000` (ou le port indiqué) dans votre navigateur.

## 🎨 Personnalisation

### Modifier les Informations Personnelles

1. **Nom & Titre** : Éditer la section hero dans `index.html`
2. **Description** : Modifier les textes dans les sections About
3. **Projets** : Remplacer les projets placeholder par vos vrais projets
4. **Compétences** : Ajuster les barres de progression dans la section Skills
5. **Contact** : Mettre à jour l'email et le téléphone dans la section Contact

### Modifier les Couleurs

Éditer les variables CSS dans `styles.css` :

```css
:root {
    --primary-color: #2563eb;      /* Couleur principale */
    --secondary-color: #3b82f6;    /* Couleur secondaire */
    /* ... autres variables */
}
```

### Modifier l'Animation de Typing

Éditer le tableau `phrases` dans `script.js` :

```javascript
const phrases = [
    'Problem Solver',
    'Full Stack Developer',
    'System Architect',
    'Technical Consultant'
];
```

### Ajouter des Technologies

Pour ajouter une nouvelle technologie avec icône et couleur :

1. Ajouter l'icône SVG dans le HTML
2. Ajouter la classe CSS `tech-[nom]`
3. Définir les couleurs dans `styles.css`
